# Учебный проект по Java: Учёт котиков и их владельцев

## Описание проекта

Данный учебный проект разработан для демонстрации принципов разработки приложений на Java с использованием Spring Boot. Проект представляет собой систему учёта котиков и их владельцев. В нем реализована архитектура MVC (Model-View-Controller) с использованием паттерна контроллер-сервис-репозиторий (Controller-Service-Data), а также активно используются возможности Spring Boot.

## Особенности проекта

- **Архитектура Controller-Service-Data:** Контроллеры (Controller) отвечают за обработку HTTP запросов и передачу данных между клиентом и сервером. Сервисы (Service) содержат бизнес-логику приложения и взаимодействуют с репозиториями для доступа к данным. Репозитории (Data) используются для взаимодействия с базой данных с помощью JPA (Java Persistence API).

- **Использование Spring Boot:** Проект разработан с использованием Spring Boot, что облегчает настройку и развертывание приложения. Встроенный контейнер сервлетов позволяет запускать приложение без необходимости конфигурирования внешних серверов приложений.

- **Обработка HTTP запросов и валидация данных:** Контроллеры умеют обрабатывать HTTP запросы от клиентов и осуществлять валидацию входных данных. Приложение предусматривает обработку различных типов запросов для управления данными о котиках и их владельцах.

## Технологии

- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Hibernate
- Валидация данных

## Примечание

Данный проект является учебным и разработан для демонстрации принципов работы с Java и Spring Boot.
